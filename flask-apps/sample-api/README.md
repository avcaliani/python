## API Usage
```sh
# Return a task list
curl http://localhost:5000/tasks

# Return a specific task
curl http://localhost:5000/tasks/1

# Create new Task
curl -d '{ "title": "New Task", "description": "Awesome \\o/" }' \
     -H 'Content-Type: application/json' \
     -X POST http://localhost:5000/tasks
```
